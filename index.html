Tendo como aprendizado:

Java:

Java é uma linguagem de programação versátil, conhecida por sua portabilidade e orientação a objetos. Criada pela Sun Microsystems, Java é executada em várias plataformas por meio da JVM, sendo ideal para desenvolvimento de aplicativos de desktop, web e móveis. Sua biblioteca padrão oferece recursos abrangentes. Além disso, Java é essencial para o desenvolvimento de aplicativos Android, alcançando um amplo público. Em resumo, Java é uma linguagem escalável e confiável que desempenha um papel central na computação moderna, tornando-se uma habilidade valiosa para programadores.

HTML:

O HTML (Hypertext Markup Language) é a linguagem essencial para criar páginas da web. Usando tags e elementos, o HTML estrutura o conteúdo online, permitindo a criação de links, imagens, parágrafos e muito mais. Essa linguagem simples e acessível é a base de todos os sites que visitamos. Desenvolvedores web dependem do HTML para criar estruturas sólidas de páginas. Além disso, o HTML5, a versão mais recente, introduziu recursos avançados, como áudio e vídeo incorporados, tornando-o ainda mais poderoso. Combinado com CSS para estilos e JavaScript para interatividade, o HTML forma a tríade fundamental do desenvolvimento web. Em resumo, o HTML é a linguagem-chave que possibilita a construção da web moderna, sendo uma habilidade crucial para qualquer pessoa envolvida no desenvolvimento ou na criação de conteúdo na internet.

CSS:

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site

Javascript:

JavaScript é uma linguagem de programação front-end, isto é, uma linguagem que interpretada diretamente no cliente (navegador). Nos últimos anos, JavaScript vem se destacando como a linguagem preferida entre todos os programadores web

SQL:

SQL (Structured Query Language) é a linguagem padrão para gerenciamento de bancos de dados relacionais. Utilizada para criar, consultar, atualizar e excluir dados, o SQL é uma ferramenta essencial para administradores e desenvolvedores de bancos de dados. Sua sintaxe poderosa e versatilidade o tornam uma escolha fundamental para lidar eficientemente com informações em aplicativos empresariais. Variantes populares, como MySQL, PostgreSQL e Microsoft SQL Server, atendem a diferentes necessidades, tornando o SQL uma habilidade indispensável para profissionais de tecnologia.
